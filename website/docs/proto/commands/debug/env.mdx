The `proto debug env` command will print information about your current proto environment. Primarily
the store location, relevant file paths, and environment variables.

```
$ proto debug env

Store
  Root: ~/.proto
  Bins: ~/.proto/bin
  Shims: ~/.proto/shims
  Plugins: ~/.proto/plugins
  Tools: ~/.proto/tools
  Temp: ~/.proto/temp
  Virtual:
    /userhome - ~
    /proto - ~/.proto
    /workspace - ~/Projects/example
  Configs:
    - ~/Projects/example/.prototools
    - ~/.proto/.prototools

Environment
  Proto version: 0.26.0
  Operating system: macos
  Architecture: arm64
  Variables:
    PROTO_HOME - ~/.proto
    PROTO_VERSION - 0.26.0
```

### Options

- `--json` - Print the list in JSON format.
