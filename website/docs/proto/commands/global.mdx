---
title: global
---

The `proto global <tool> <version>` command will set the global default version of a tool, by
creating a version file in the `~/.proto/tools/<tool>` installation directory. This version will be
used when attempting to detect a version in [`proto bin`](./bin) and [`proto run`](./run).

```shell
$ proto global go 1.20.1
```

### Arguments

- `<tool>` - Type of tool.
- `<version>` - Version of tool.
