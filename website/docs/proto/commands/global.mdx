---
title: global
---

The `proto global <tool> <version>` command will set the global default version of a tool, by
creating a version file at `~/.proto/tools/<tool>/version`. This version will be used when
attempting to [detect a version](../detection).

```shell
$ proto global go 1.20.1
```

### Arguments

- `<tool>` - Type of tool.
- `<version>` - Version of tool.
