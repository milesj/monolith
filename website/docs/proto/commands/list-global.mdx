---
title: list-global
---

The `proto list-global <tool>` (or `proto lsg`) command will list installed global dependencies (and
their canonical file location) by scanning the file system at the tool's global bins directory.

```shell
$ proto list-global go
gopls - ~/go/bin/gopls
mockery - ~/go/bin/mockery
reflex - ~/go/bin/reflex
```

:::info

This command lists all files within the globals directory, and may include files that were not
installed with [`proto install-global`](./install-global).

:::

### Arguments

- `<tool>` - Type of tool.
