The `proto plugin remove <tool>` command (formerly `proto remove-plugin`) will remove the provided
tool ID from the `[plugins]` section of the local (`.prototools`) or global (`~/.proto/.prototools`)
config file.

```shell
$ proto plugin remove node
```

> Built-in plugins _cannot_ be removed!

### Arguments

- `<tool>` - ID of the tool.

### Options

- `--global` - Pin the version globally, rather than in the current directory.
